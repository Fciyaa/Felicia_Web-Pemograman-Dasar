1. Heading

HTML headings digunakan untuk memberi struktur dan hierarki pada konten halaman web. Heading adalah elemen teks yang menunjukkan judul atau subjudul dari suatu bagian di halaman. Ada enam level heading yang bisa digunakan, dari h1 hingga h6, dengan h1 sebagai yang paling penting atau terbesar, dan h6 sebagai yang terkecil atau paling tidak penting.

2. Paragraphs

HTML paragraf digunakan untuk mengelompokkan teks dalam sebuah bagian atau kalimat yang lebih panjang. Paragraf dibungkus dengan elemen p, yang secara otomatis memberi jarak antara teks di paragraf tersebut dengan elemen lain di sekitarnya.

Penjelasan:

- p adalah elemen pembuka (start tag) untuk paragraf.
- /p adalah elemen penutup (end tag) untuk paragraf.
- Teks yang ada di dalam tag p akan ditampilkan sebagai paragraf di halaman web.

3. Styles

HTML styles (gaya) digunakan untuk menentukan bagaimana elemen-elemen di halaman web akan ditampilkan kepada pengguna. Gaya ini bisa mencakup pengaturan warna, font, ukuran teks, tata letak, dan banyak lagi. Ada beberapa cara untuk menerapkan gaya pada elemen HTML, dan ini disebut dengan CSS (Cascading Style Sheets).

Ada tiga cara untuk menambahkan gaya (style) pada HTML:

- Inline CSS: Menambahkan gaya langsung ke elemen HTML.
Inline CSS adalah cara menambahkan gaya langsung di dalam tag HTML menggunakan atribut style. Metode ini hanya mempengaruhi elemen yang diberi atribut style
- Internal CSS: Menyertakan gaya di dalam tag style yang ada di dalam elemen head pada halaman HTML.
Internal CSS digunakan dengan menambahkan kode gaya di dalam tag style di bagian head dari halaman HTML. Gaya ini berlaku untuk seluruh elemen yang sesuai dengan aturan CSS yang ditulis.
- External CSS: Menggunakan file CSS terpisah dan menghubungkannya ke file HTML.
External CSS adalah cara paling efektif dan paling fleksibel untuk mengelola gaya di halaman web. Dalam metode ini, kita membuat file CSS terpisah dan kemudian menghubungkannya ke file HTML dengan menggunakan tag link.

5. Quotations

HTML menyediakan elemen khusus untuk menandai kutipan (quotation) dan kutipan sumber (citation) di dalam teks, yang sangat berguna untuk memberikan konteks dan atribusi yang tepat dalam konten berbasis teks di halaman web. Elemen-elemen ini juga dapat membantu mesin pencari memahami struktur halaman dan memberikan informasi tambahan kepada pembaca.

Elemen-elemen

- Blockquote Element

Element <blockquote> digunakan untuk menandai kutipan panjang dari sumber lain:

For 60 years, WWF has worked to help people and nature thrive. As the world's leading conservation organization, WWF works in nearly 100 countries. At every level, we collaborate with people around the world to develop and deliver innovative solutions that protect communities, wildlife, and the places in which they live.
- Q Element

Element <q> digunakan untuk kutipan pendek dalam paragraf:

WWF's goal is to: Build a future where people live in harmony with nature.

- Abbreviation Element

Element <abbr> digunakan untuk menandai singkatan atau akronim:

The WHO was founded in 1948.

- Address Element

Element <address> digunakan untuk informasi kontak:

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA
- Cite Element

Element <cite> digunakan untuk menandai judul karya kreatif:

The Scream by Edvard Munch. Painted in 1893.

- BDO Element

Element <bdo> digunakan untuk mengganti arah teks:

This text will be written from right to left

8. CSS

HTML (HyperText Markup Language) dan CSS (Cascading Style Sheets) adalah dua komponen utama dalam pengembangan web yang digunakan untuk membuat halaman web yang terstruktur dan menarik. Meskipun keduanya memiliki peran yang berbeda, mereka bekerja bersama-sama untuk membangun dan mendesain halaman web.

Using CSS:

- Inline CSS (menggunakan atribut style pada elemen HTML)
- Internal CSS (menggunakan elemen <style> di bagian <head>)
- External CSS (menggunakan file CSS eksternal)
